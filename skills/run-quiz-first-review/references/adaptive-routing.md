# Adaptive review routing

Use session evidence to choose the smallest useful next step.

## Response labels

- **Correct:** Includes the core idea and any source qualifier required to make it accurate.
- **Partly correct:** Shows the core direction but omits, confuses, or overstates a meaningful part.
- **Incorrect:** Misses the central relationship, contradicts the source, or gives no answer.

## Routing

- After **correct**, select a different single source clause and ask one direct-retrieval question. Ask a comparison, sequence, or application only when one supplied sentence explicitly states the entire requested relationship.
- After **partly correct**, state the smallest missing idea and ask a narrower retrieval check on that idea.
- After **incorrect**, quote or closely paraphrase only the selected source clause, then ask a direct-retrieval question from another explicit clause before returning later.
- After two misses on the same idea, stop escalating. Give a compact explanation plus an example already present in the source.
- Revisit missed ideas after at least one intervening topic when the question limit allows.

## Hint ladder

Give only one hint level at a time.

1. **Source location:** Name an explicit learner-supplied heading, page, slide, or sentence number when available; otherwise say `Look at the source sentence used for this question.`
2. **Exact-word cue:** Quote a short contiguous fragment immediately beside the retrieval target without adding a category or relationship label.
3. **Blanked clause:** Quote the selected source clause with only the answer span replaced by a blank.
4. **Answer plus check:** Quote or closely paraphrase that one source clause, then ask the learner to restate the same clause in their own words.

Do not present the hidden answer key or future questions as part of a hint.
